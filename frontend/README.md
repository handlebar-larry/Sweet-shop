# SweetShop

Sweet shop management, simplified.

## What you can do
- Create an account and sign in
- Add, edit, and delete sweets
- Upload an image for each sweet
- Search and filter by name / category / price
- Purchase sweets and keep stock updated

## Run locally
```bash
npm install
npm run dev
```

## Environment variables
Create a `.env` file in the project root and set your API URL:

```env
VITE_BACKEND_URL=http://localhost:3500
```

(If your backend runs on a different port or domain, update the value above.)
